# User Flow

## Authentication Flow
Register -> Login -> Dashboard

## Meal Tracking Flow
Dashboard -> Today's Meals -> Add Meal -> Search Food (API) -> Select Food -> Set Serving -> Save Meal -> Return to Dashboard

## Exercise Tracking Flow
Dashboard -> Today's Exercise -> Add Exercise -> Select Activity (API) -> Set Duration -> Save Exercise -> Return to Dashboard

## Water Tracking Flow
Dashboard -> Water Tracker -> Select Cup(s) -> Update Daily Water Intake -> Refresh Dashboard Summary

## Calendar Flow
Dashboard -> Calendar -> Select Date (Last 7 Days) -> View Daily Summary

## Blog Flow
Dashboard -> Blog Preview -> Blog Listing -> Blog Detail
